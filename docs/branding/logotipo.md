O logótipo e o ícone da AIDE são os elementos centrais da identidade visual da marca e devem ser utilizados de forma consistente em todas as suas aplicações.

## Área de Proteção

O logótipo deve sempre manter uma área de proteção mínima ao seu redor para garantir visibilidade e impacto. Esta área corresponde à altura do logótipo.

```
┌─────────────────────────┐
│         ↕ X             │
│    ┌───────────┐        │
│ ←X │   AIDE    │ X→     │
│    └───────────┘        │
│         ↕ X             │
└─────────────────────────┘
```

---

## Tamanhos Mínimos

Para garantir legibilidade, o logótipo não deve ser reproduzido abaixo dos seguintes tamanhos:

| Aplicação | Tamanho Mínimo |
|-----------|----------------|
| **Digital** | 80px de largura |
| **Impressão** | 20mm de largura |

---

## Aplicação com Fundos

É importante colocar sempre o logótipo em fundos com bom contraste para garantir a sua legibilidade.

### Diretrizes de Aplicação

- **Fundos fotográficos**: Utilize preferencialmente a versão clara (branca) do logótipo sobre imagens escuras
- **Áreas de baixa luminosidade**: Posicione o logótipo em zonas da imagem com menos detalhe visual
- **Contraste mínimo**: Garanta sempre contraste suficiente entre o logótipo e o fundo
- **Consistência**: Mantenha o logótipo e a tagline "O futuro do cuidado é conectado" alinhados e legíveis

### Boas Práticas

| Situação | Recomendação |
|----------|--------------|
| **Fundo escuro uniformes** | Versão clara (branca) do logótipo |
| **Fundo claro uniforme** | Versão escura (preta) do logótipo |
| **Imagens fotográficas** | Avaliar contraste; preferir zonas escuras para logótipo claro |
| **Fundos complexos** | Adicionar área de proteção semi-transparente se necessário |

---

## Uso Incorreto

:::caution O que NÃO fazer
- Distorcer ou esticar o logótipo
- Alterar as cores do logótipo
- Adicionar sombras, bordas ou efeitos
- Colocar sobre fundos com pouco contraste
- Rodar o logótipo
- Usar versões de baixa resolução
:::
