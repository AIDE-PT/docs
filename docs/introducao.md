# O que é a AIDE?

A AIDE é uma solução tecnológica concebida para dotar os cuidadores de um painel de controlo centralizado para a gestão da segurança, rotinas e saúde no domicílio. O projeto assenta na premissa de que cada cenário de cuidado é único; por isso, a aplicação não impõe uma estrutura rígida, mas oferece antes um ecossistema modular e adaptável às necessidades específicas de cada utilizador.

## Quem faz a AIDE acontecer?

A AIDE é desenvolvida por uma equipa dedicada e apaixonada por melhorar a vida de todos.
